# Bach Portfolio (porto-bach)

This repository serves as a portfolio for my website projects and a collection of specialized content processing utilities.

## 🚀 Featured Apps

Inside the `simple apps` directory, you will find several Python-based tools designed for scraping, content conversion, and automation:

### 📥 Downloaders
- **[Skool Downloader](simple%20apps/skool-downloader/)**: Automates downloading community content using Selenium.
- **[Blog Scraper](simple%20apps/getblog/)**: Fetches and cleans articles from web URLs.

### 🔄 Converters
- **[HTML to Markdown](simple%20apps/html2md/)**: Converts HTML files into clean Markdown, specialized for Skool content.
- **[PGA Blog to Markdown](simple%20apps/htmltomd-pga/)**: Optimized conversion for the Premium Ghostwriting Academy blog.
- **[Markdown to PDF](simple%20apps/mdtopdf/)**: Generates professionally styled PDFs from Markdown files.

### 🛠 Utilities
- **[Video Link Extractor](simple%20apps/vidlink/)**: Extracts and catalogs video links (Loom, YouTube, etc.) into CSV format.
- **[Markdown Combiner](simple%20apps/md-combiner/)**: Merges multiple Markdown files into a single master document.

## 🌐 Web Portfolio
The main `index.html` in the root directory contains the landing page for my portfolio.
