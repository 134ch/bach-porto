# Bach Portfolio (porto-bach)

This repository serves as a portfolio for my website projects and a collection of specialized content processing utilities.

## 🚀 Featured Apps

Inside the `simple apps` directory, you will find several Python-based tools designed for scraping, content conversion, and automation:

### 📥 Downloaders
- **[Skool Downloader](simple%20apps/README_skool_downloader.md)**: Automates downloading community content using Selenium.
- **[Blog Scraper](simple%20apps/README_getblog.md)**: Fetches and cleans articles from web URLs.

### 🔄 Converters
- **[HTML to Markdown](simple%20apps/README_html2md.md)**: Converts HTML files into clean Markdown, specialized for Skool content.
- **[PGA Blog to Markdown](simple%20apps/README_htmltomd_pgablogonly.md)**: Optimized conversion for the Premium Ghostwriting Academy blog.
- **[Markdown to PDF](simple%20apps/README_mdtopdf.md)**: Generates professionally styled PDFs from Markdown files.

### 🛠 Utilities
- **[Video Link Extractor](simple%20apps/README_Vidlink.md)**: Extracts and catalogs video links (Loom, YouTube, etc.) into CSV format.
- **[Markdown Combiner](simple%20apps/README_md_combiner.md)**: Merges multiple Markdown files into a single master document.

## 🌐 Web Portfolio
The main `index.html` in the root directory contains the landing page for my portfolio.
